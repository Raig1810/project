# Описание проекта: Разработка CRUD-сервиса на Python и FastAPI с контейнеризацией

---

## Назначение проекта: REST API сервис на FastAPI для управления базой данных студентов 


## Версии используемых технологий:
* Python 3.11
* FastAPI 0.139.0
* Uvicorn 0.51.0
* SQLAlchemy 2.0.51
* Pydantic 2.13.4
* Loguru 0.7.3

---

## Основные эндпоинты

* / – корневой маршрут;
* /health – проверка работоспособности и стабильности сервиса;
* /create – создание новой записи о студенте;
* /read – получение данных (по id или списка всех активных студентов, с возможностью фильтрации по группе);
* /update/{student_id} – изменение существующей записи;
* /delete/{student_id} – мягкое удаление записи (перевод флага is_active в False).

---

## Инструкция по запуску 

 Для запуска требуется установленный Docker

 Открыть терминал в корневой директории проекта ('project/')
 
 Ввести команду для запуска FastAPI сервиса в терминал:
 ```bash
 docker compose up --build
 ```

## Примеры запросов к API 

Пример создания записи: 

```json
{
  "name": "Иван Иванов",
  "group": "ИБ-251",
  "average_score": 85.5
}
```

Пример изменения существующей записи:

```json
{
  "new_name": "Иван Иванов",
  "new_score": 90.2
}
```

Пример ответа при запросе списка студентов

```json
{
  "id": 1,
  "name": "Алексей Смирнов",
  "group": "ИВТ-11",
  "average_score": 92.0,
  "is_active": true
}
```
